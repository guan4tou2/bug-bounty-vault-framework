# Post-Run Knowledge Capture

## Purpose

Make sure reviewed work improves the private knowledge base instead of leaving only raw output behind.

## Trigger

After a recon session, automation run, review note, or finding update.

## Inputs

- Run summary.
- Candidate list.
- Review decisions.
- Notes that may contain reusable lessons.

## Stop conditions

- The run has no reviewed output.
- The lesson depends on private target data that cannot be generalized.
- Raw output, secrets, or screenshots would be copied into durable notes.

## Output

Return suggested Pattern, Playbook, Checklist, or Reference Card updates, plus any private details that must remain out of public material.
