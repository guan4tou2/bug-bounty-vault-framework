# Knowledge Capture Skill

Use when converting completed work into reusable private knowledge.

## Authorized scope

Only process notes from authorized work. Keep target-specific data private.

## Allowed inputs

- Recon summary.
- Finding summary.
- Triage result.
- Lessons or decision notes.

## Process

1. Extract reusable pattern.
2. Identify target-specific details that must stay private.
3. Create a public-safe abstraction.
4. Recommend private wiki updates.

## Stop conditions

- Source material is unverified.
- The lesson cannot be generalized safely.
- The output would reveal private target data.

## Output

Return reusable lesson, private details to retain, safe abstraction, and update checklist.
