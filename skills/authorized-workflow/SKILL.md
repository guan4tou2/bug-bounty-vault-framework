# Authorized Workflow Skill

Use when starting, reviewing, or closing an authorized security research workflow.

## Authorized scope

This skill requires an explicit scope note or placeholder scope. If scope is absent, stop before planning active work.

## Allowed inputs

- Scope summary.
- Target placeholder.
- Existing notes.
- Requested workflow step.

## Process

1. Confirm scope.
2. Check for duplicates.
3. Select the next safe workflow gate.
4. Record evidence requirements.
5. Prepare knowledge capture.

## Stop conditions

- Scope is unclear.
- The request asks for unauthorized action.
- Evidence collection would expose sensitive unrelated data.

## Output

Return scope status, selected gate, required evidence, and next action.
