# Vault Tool Notes

This folder stores durable notes about tool usage, helper scripts, and repeatable configuration decisions that belong in the Obsidian vault.

It is not the runtime toolchain. Scanner repos, downloaded binaries, large outputs, and generated logs belong in the ignored `workspace/` runtime area or in a user-managed external tools directory.

Use this layer for:

- Tool selection notes
- Reusable command patterns
- Configuration rationale
- Links to public documentation
- Lessons that are too tool-specific for `09 - Knowledge Base/`
